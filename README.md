# 🟣 jelly-slime

## ✨ Features

* **Spring-mass soft body** — physics implemented with Matter.js `Bodies` + `Composites.softBody()` constraints.
* **Mouse drag → stretch** — grab any node, drag hard, release: watch the slime bounce back like jelly.
* **Wall collisions → squash & jiggle** — a viscous damping + structural constraints reshape the blob on impact.
* **Visible internal mesh** — every spring/constraint is drawn so you can *see* the soft-body structure working.
* **Neon glow** — additive blending + layered shadows for that candy-glow look.
* **Zero runtime deps** — Matter.js is inlined via CDN pin inside the single `index.html`. Offline-friendly after first load.

## 🚀 Run it locally

```bash
git clone https://github.com/sigco3111/jelly-slime.git
cd jelly-slime
open index.html        # macOS
# xdg-open index.html  # Linux
# start index.html     # Windows
```

That's it. No `npm install`, no build step. First load pulls Matter.js from the CDN pinned in `index.html`.

> 🛜 **Offline?** Open the page once online so the cache holds Matter.js. After that the demo works without a network.

## 🛠 Make it your own — three sample riffs

### 1) Ghostly green slime (one-line tint change)
```html
<style>body{background:#0a0a14}</style>
```
In `index.html` change the primary color from `#ff3df5` → `#39ff14`, and re-run.

### 2) Toggle the skeleton off (clean outline look)
Find `renderSkeleton = true` in `index.html`, flip to `false`. The slime becomes a smooth outline-only shape.

### 3) Stiffer jelly — bouncier returns
Bump the `stiffness` argument passed into `Composites.softBody` from `0.05` to `0.2`. The blob snaps back faster.
